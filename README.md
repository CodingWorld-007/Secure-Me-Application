# SecureMe 🔐

A privacy-first encrypted vault for storing passwords, bank details, crypto wallets, and sensitive credentials securely on Android.

Built with **Kotlin + Jetpack Compose** and designed with **security-first architecture**.

---

### Tech Stack

- Kotlin
- Jetpack Compose
- SQLCipher encrypted database
- AES encryption
- Android Biometric API
- MVVM architecture

---

### Features

• Biometric authentication (fingerprint / device security)  
• AES encrypted local vault  
• Recovery key system for account access  
• Screenshot protection for sensitive screens  
• Secure storage for passwords, bank accounts, crypto wallets, and cards  
• Offline-first design — no cloud storage

Your data **never leaves your device.**
